## 文件结构
```
src/
├── assets/              # 静态资源（图片、字体）
├── components/          # 通用组件库（按钮等）
├── contexts/            # 状态管理
├── features/            # 各页面模块
├── hooks/               # 自定义Hooks（主题切换等）
├── router/              # 路由配置
├── services/            # 服务层
├── styles/              # 全局样式与CSS变量
├── utils/               # 工具方法集合
```

## 开发

安装所有依赖
```
npm install
```

本地运行
```
npm run dev
```

提交（先git add，建议使用npm指令提交统一commit格式）
```
npm run commit
```
